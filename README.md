# My Wayland configuration

### Sway 
```
yay -S sway
```

### Utilities
```
// kitty - terminal
yay -S kitty

// dunst for notification
yay -S dunst 

// waybar as bar
yay -S waybar

// rofi - application launcher
yay -S rofi-lbonn-wayland

// wlsunset - night light
yay -S wlsunset

// brightnessctl - backlight controler
yay -S brightnessctl

// amixer - volume control
yay -S amixer

// clipmenu - clipboard manager
yay -S clipmenu
